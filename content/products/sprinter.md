### Get Started
```terminal
    # binary is created on the go/bin directory.
    go get -u github.com/gari8/sprinter
```

### Command
```terminal
    # Interactive interface launch.
    sprinter -n or -new
    
    # It show help
    sprinter -h or -help
```
